# 체스 서버 프로젝트

## 시작하기


1. 필요한 패키지 설치:
    ```sh
    sudo apt update
    sudo apt install cmake build-essential
    ```

2. 프로젝트 디렉토리에서 `build` 폴더 생성:
    ```sh
    mkdir build
    cd build
    ```

3. CMake를 사용하여 빌드 시스템 생성:
    ```sh
    cmake ..
    ```

4. 프로젝트 빌드:
    ```sh
    make
    ```

5. 서버 실행:
    ```sh
    ./chess_server
    ```
